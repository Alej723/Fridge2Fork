Fridge2Fork Demo - Backup

1. Problem & Solution

Problem: Food waste, meal planning stress, dietary restrictions

Solution: Fridge2Fork turns ingredients into meal plans

2. Key Features

  • Search recipes by ingredients you have
  • Filter by dietary preferences (Halal, Vegetarian, etc.)
  • Weekly meal planner with drag-and-drop
  • Automatic grocery list generation
  • Print/download shopping lists

3. Tech Stack

  • Frontend: React with responsive design
  • Backend: Node.js/Express with JWT auth
  • Database: MongoDB for user data
  • APIs: Spoonacular for recipes
